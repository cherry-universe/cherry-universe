自带 好的，那么现在我们安装完了flatpak，一起来看看怎么愉快(?)地使用吧:

镜像源配置

flatpak remote-add --if-not-exists --user flathub https://dl.flathub.org/repo/flathub.flatpakrepo flatpak remote-modify flathub --url=https://mirrors.ustc.edu.cn/flathub

使用

安装软件包

flatpak install [-y] flathub xxx

xxx是要安装软件包的名称。如果要安装某个冷门的软件包的话，你大概率会发出愉悦的尖锐爆鸣 因为镜像站没缓存，直接给你重定向到flathub官网了

卸载软件包

flatpak uninstall [--delete-data] [-y]

--delete-data是把用户数据一块删掉

卸载未使用的运行时

flatpak uninstall --unused

查看已安装的包列表

flatpak list

更新所有软件包

flatpak update
